# План работ по сайту «Английский Без Барьера»

## Итерация 1 (реализована)

1. Переработан контент главной страницы по брифу клиента:
- позиционирование, ценности и отличия;
- форматы занятий (очно/онлайн, индивидуально/пара/мини-группа);
- уровни CEFR + Young Learners (Starters, Movers, Flyers);
- FAQ под реальные вопросы родителей.

2. Обновлён блок цен:
- разовая оплата: 3000 / 2500 / 2000 ₽;
- абонемент: 15200 / 13680 / 11400 ₽;
- зафиксирована длительность занятия 55 минут и пробное бесплатно.

3. Подключены медиа клиента:
- логотип;
- фирменные фоны;
- реальные кадры с занятий в галерее и карточках материалов.

4. Подготовлена интеграция LeadPay:
- API-роут генерации платёжной ссылки;
- перевод оформления товара на реальный переход в LeadPay;
- fallback-режим через статичную страницу оплаты.

5. Добавлена страница успешной оплаты:
- `/payment/success`;
- инструкции для покупателя по получению PDF и чека.

## Итерация 2 (в работе)

1. Визуальная переработка под фирменный стиль:
- палитра в цветах клиента (графит, тёплые пастели, красно-оранжевый акцент);
- новый hero с атмосферным motion-эффектом;
- логотип без фона в шапке и футере;
- замена favicon и системных иконок на фирменные.

2. Раздел «Материалы» усилен как отдельная коммерческая зона:
- уточнено, что материалы можно приобрести сразу;
- добавлен явный сценарий покупки;
- добавлено уточнение про самовывоз из студии для печатных комплектов.

3. Медиа в каталоге и галерее:
- обновлены превью под каждую карточку материалов;
- заменены слабые/неудачные кадры;
- использованы релевантные изображения для учебных карточек, игр и рабочих материалов.

4. Контент:
- тексты скорректированы в более профессиональной и тёплой подаче;
- убраны формулировки-заглушки.

5. Следующий шаг итерации:
- расширение каталога до 50–70 товарных позиций;
- категоризация по темам и уровням;
- автоматическая выдача PDF через postback/webhook после успешной оплаты.

## Итерация 3

1. SEO/GEO-посадочные страницы под приоритетные запросы.
2. Локальная микроразметка и усиление блоков доверия.
3. Финальное наполнение отзывами, блоком владельца и медиа.
4. Подготовка к прод-развёртыванию.
